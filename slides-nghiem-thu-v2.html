07 / 12
BUG-01
GAS URL mismatch — config.js
⚠️ Nguyên nhân: URL trỏ sai deployment cũ + nested quotes syntax error
✅ Fix: Rewrite config.js với URL deployment mới, dùng single quotes
BUG-02
Script 404 — không tải được JS/CSS
⚠️ Nguyên nhân: index.html load scripts từ aifun-skill-engine/ không tồn tại
✅ Fix: Sửa 4 path về root directory
BUG-03
getSkillForm is not defined
⚠️ Nguyên nhân: skill-forms.js bị truncate tại line 39, thiếu function + window exports
✅ Fix: Viết lại hoàn chỉnh 6 skills + window.SKILL_FORMS + window.getSkillForm
BUG-04
Submit button không click được (Cốc Cốc)
⚠️ Nguyên nhân: type=submit + form onsubmit không fire trong Cốc Cốc browser
✅ Fix: Đổi sang type=button + onclick="SkillEngine._onFormSubmit()"
BUG-05
Cannot read properties of undefined (reading 'map')
⚠️ Nguyên nhân: form tồn tại nhưng aiOutputSections undefined → undefined.map() crash
✅ Fix: Guard (form && form.aiOutputSections) ? ... : fallback
Bug Fixes · 2/2
10 lỗi đã xử lý
CORS · Claude API · Logger · Stats · Word Export
08 / 12
BUG-06
CORS "Failed to fetch"
⚠️ Content-Type: application/json → trigger OPTIONS preflight → GAS không handle
✅ Fix: Dùng Content-Type: text/plain (simple request, không cần preflight). GAS tự add CORS header.
BUG-07
Claude API 404 — model not found
⚠️ AIProvider.gs dùng claude-3-5-sonnet-20241022 đã deprecated
✅ Fix: Cập nhật CLAUDE_MODEL = 'claude-sonnet-4-6' → deploy GAS mới
BUG-08
Logger.gs không ghi Sheets — silent fail
⚠️ SpreadsheetApp.openById() throw "Illegal spreadsheet id" bên trong try-catch → lỗi bị nuốt, GAS vẫn return success
✅ Fix: Dùng DriveApp.getFilesByName('AIFUN DATA CENTER') + SpreadsheetApp.open(file)
BUG-09
CEO Dashboard hiển thị 0
⚠️ GAS getStats đọc sai index cột do column layout thực tế khác thiết kế ban đầu
✅ Fix: Tính stats client-side từ getDocuments + _computeStats() với _parseDate()
BUG-10
Word Export lỗi với nội dung dài
⚠️ Content truyền inline qua onclick attribute → bị cắt / break với ký tự đặc biệt tiếng Việt
✅ Fix: Lưu content vào window._seLastContent, _downloadDocx() đọc từ global — HTML-Word format
